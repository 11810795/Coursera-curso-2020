We would like to find a series for \(f(t) = \int_0^t e^{-x^2} \, dx\).

One helpful thing is that \(e^x = \sum_{n=0}^\infty \frac{x^n}{n!}\).

This is a tricky step: you might then believe that \(e^{-x^2} = \sum_{n=0}^\infty \frac{\left(-x^2\right)^n}{n!}\).

So we can simplify this to get \(e^{-x^2} = \sum_{n=0}^\infty \frac{(-1)^n \, x^{2n}}{n!}\).

That looks a bit like an alternating series! That is a good sign (!).

Another thing we can do is integrate a power series term-by-term.

That means that \(\int_{x=0}^t e^{-x^2} \, dx = \int_{x=0}^t \sum_{n=0}^\infty \frac{(-1)^n \, x^{2n}}{n!} dx = \sum_{n=0}^\infty \int_{x=0}^t \frac{(-1)^n x^{2n}}{n!} dx\).

But \(\int_{x=0}^t x^{2n} \, dx = \frac{x^{2n+1}}{2n+1}\).

So \(f(t) = \int_{x=0}^t e^{-x^2} \, dx = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{n! \cdot (2n+1)}\).

We want to approximate \(f\left(#{latex(desired_point)}\right) = \sum_{n=0}^\infty \frac{(-1)^n \left(#{latex(desired_point)}\right)^{2n+1}}{n! \cdot (2n+1)}\).

This is an alternating series, so we know how to approximate the error in a partial sum.

We look for a term in the series with absolute value no larger than \(#{latex(goal_accuracy)}\).

So we can talk about this more easily, set \(a_{#{variable}} = #{latex(term)}\).

Plugging in a few values, we find that when \(#{variable} = #{latex(need_terms+1)}\), then \(|a_{#{need_terms+1}}| = #{latex(abs(term_x(x=need_terms+1)))}\) which is less than \(#{latex(goal_accuracy)}\).

So the true value of the series is between \(\sum_{#{variable} = #{initial}}^{#{latex(need_terms)}} #{latex(term)}\) and \(\sum_{#{variable} = #{initial}}^{#{latex(need_terms+1)}} #{latex(term)}\).

Doing some arithmetic, \(\sum_{#{variable} = #{initial}}^{#{latex(need_terms)}} #{latex(term)} = #{latex(sum(term_x,var('x'),initial,need_terms))}\).

Doing some more arithmetic, \(\sum_{#{variable} = #{initial}}^{#{latex(need_terms+1)}} #{latex(term)} = #{latex(sum(term_x,var('x'),initial,need_terms+1))}\).

Looking over the possible choices, \(#{latex(rational_answer)}\) is between \(#{latex(sum(term_x,var('x'),initial,need_terms))}\) and \(#{latex(sum(term_x,var('x'),initial,need_terms+1))}\).
