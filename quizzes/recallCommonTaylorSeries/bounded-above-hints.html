The definition begins by asserting the existence of some bound \(M\).

The bound is a real number, meaning \(M \in \mathbb{R}\).

So the definition begins “there exists an \(M \in \mathbb{R}\)…”

The bound must hold for all terms in the sequence.

So we will assert something for all indexes \(n \in \mathbb{N}\).

The definition begins “there exists an \(M \in \mathbb{R}\), so that for all \(n \in \mathbb{N}\), …”

For a particular term \(a_n\) to be bounded above by \(M\) just means that \(a_n \leq M\).

Altogether then, being bounded above means “there exists an \(M \in \mathbb{R}\), so that for all \(n \in \mathbb{N}\), we have \(a_n \leq M\).” Sometimes you might see this ending with \( a_n < M \), but that is a difference which does not affect which sequences are bounded above.
