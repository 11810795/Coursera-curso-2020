We should apply Taylor's theorem.

Let's start writing the Taylor series for \(f\) around the point \(#{latex(the_center)}\).

This is \(f(x) = f(#{latex(the_center)}) + f'(#{latex(the_center)})\,(x-#{latex(the_center)}) + \frac{f^{(2)}(#{latex(the_center)})}{2} \,(x-#{latex(the_center)})^2 + R_2(x)\) with some remainder term \(R_2(x)\).

Taylor's theorem gives us some control over the remainder.

So \(R_2(x) = \frac{f^{(3)}(z)}{3!} \,(x-#{latex(the_center)})^3\) for some point \(z\) between \(#{latex(the_center)}\) and \(x\).

We were asked to say something about \(f(#{latex(the_point)})\).

In this case \(f(#{latex(the_point)}) = f(#{latex(the_center)}) + f'(#{latex(the_center)})(#{latex(the_point)}-#{latex(the_center)}) + \frac{f^{(2)}(#{latex(the_center)})}{2} (#{latex(the_point)}-#{latex(the_center)})^2 + R_2(#{latex(the_point)})\).

We were told that \(f(#{latex(the_center)}) = #{latex(f_at_center)}\) and \(f'(#{latex(the_center)}) = #{latex(derivative_at_center)}\) and \(f^{(2)}(#{latex(the_center)}) = #{latex(second_derivative_at_center)}\).

Substituting these facts into the formula for \(f(#{latex(the_point)})\) gives us that \(f(#{latex(the_point)}) = #{latex(f_at_center)} + #{latex(derivative_at_center)} \, (#{latex(the_point)}-#{latex(the_center)}) + \frac{ #{latex(second_derivative_at_center)}}{2} (#{latex(the_point)}-#{latex(the_center)})^2 + R_2(#{latex(the_point)})\).

Doing some arithmetic yields \(f(#{latex(the_point)}) = #{latex(f_at_center + derivative_at_center * (the_point - the_center) + (second_derivative_at_center/2) * (the_point - the_center)**2)} + R_2(#{latex(the_point)})\).

How big and small can \(R_2(#{latex(the_point)})\) be?

Since \(R_2(x) = \frac{f^{(3)}(z)}{3!} \,(x-#{latex(the_center)})^3\), we have that \(R_2(#{latex(the_point)}) = \frac{f^{(3)}(z)}{3!} \,(#{latex(the_point)}-#{latex(the_center)})^3 = \frac{f^{(3)}(z)}{3!} \cdot #{latex((the_point - the_center)**3)}\).

We were promised that \(|f^{(3)}(x)| \leq #{latex(third_derivative_bound)}\).

Therefore \( |R_2(x)| = \left| \frac{f^{(3)}(z)}{3!} \cdot #{latex((the_point - the_center)**3)} \right| \leq \left| \frac{#{latex(third_derivative_bound)}}{3!} \cdot #{latex((the_point - the_center)**3)} \right| = #{latex(third_derivative_bound * (the_point - the_center)**3 / 6)}\).

And so \(\left|f(#{latex(the_point)}) - #{latex(f_at_center + derivative_at_center * (the_point - the_center) + (second_derivative_at_center/2) * (the_point - the_center)**2)} \right| = \left|R_2(#{latex(the_point)})\right| \leq #{latex(third_derivative_bound * (the_point - the_center)**3 / 6)}\).

Another way of writing this is \(#{latex(-abs(third_derivative_bound * (the_point - the_center)**3 / 6))} \leq f(#{latex(the_point)}) - #{latex(f_at_center + derivative_at_center * (the_point - the_center) + (second_derivative_at_center/2) * (the_point - the_center)**2)} \leq #{latex(abs(third_derivative_bound * (the_point - the_center)**3 / 6))}\).

Adding \(#{latex(f_at_center + derivative_at_center * (the_point - the_center) + (second_derivative_at_center/2) * (the_point - the_center)**2)}\) to these inequalities lets us conclude that \(#{latex((f_at_center + derivative_at_center * (the_point - the_center) + (second_derivative_at_center/2) * (the_point - the_center)**2)-abs(third_derivative_bound * (the_point - the_center)**3 / 6))} \leq f(#{latex(the_point)}) \leq #{latex((f_at_center + derivative_at_center * (the_point - the_center) + (second_derivative_at_center/2) * (the_point - the_center)**2) +abs(third_derivative_bound * (the_point - the_center)**3 / 6))}\).
